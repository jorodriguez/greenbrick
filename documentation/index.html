1. Introduction

We would like to thank you for choosing Solari Template.

We made Solari from the ground-up with flexibility in mind. Each element of Solari is extremely customizable, where you can make Solari to reflect your own branding styles.

The guide gives you detailed methodologies about how you can customize Solari and make it fit your brand perfectly!

1.1. What's Included

After purchasing Solari template on themeforest.net with your Envato account, go to your Download page. You can choose to download Solari template only or the entire Solari template package which contains the following files:

  • Solari: A HTML template file.
  • documentation: This folder contains what you are reading now.

Support Scope

Included in Free Support Scope:

  • Fixing bugs
  • Helping clients in changing site contents by sending instructions that couldn’t be covered by documentation

NOT Included in Free Support Scope:

  • Any type of custom changes
  • Any type of request to update clients’ site contents

Paid Support Scope:

Paid support is not a part of Free Support when you purchase the template from themeforest. We understand that some clients need custom changes while using our template. For that we kept an option for paid support which includes:

  • Custom change request
  • Custom pages
  • Custom features

We kept a flat rate for our paid support which is $25/hour.
